# MoveIt 2 Requirements for SemuBot

Future MoveIt 2 enablement requires:

- Accurate arm/head joint limits in URDF/xacro.
- Planning groups for left arm, right arm, and optional head.
- Collision meshes for torso/head/arms and world objects.
- ros2_control trajectory controllers for each planning group.
- Safe velocity/acceleration limits validated in lab.

This package is a placeholder until kinematic and controller data are finalized.
